# Site Web St2I.Co

Ce site web est une version simple inspirée du site de Multi-service Coulibaly Katana, adaptée pour St2I.Co, société de travaux d'ingénierie industriel et commerce.

## Structure

- `index.html` : Page principale avec header, sections Accueil, Projets, À propos, Services, Contact.
- `style.css` : Feuille de style pour le design.
- `script.js` : Script JavaScript pour le diaporama.

## Prévisualisation

Ouvrez `index.html` dans votre navigateur web pour voir le site.

## Personnalisation

- Remplacez l'image placeholder dans le header par votre photo réelle.
- Modifiez le contenu selon vos besoins.
- Ajoutez plus de sections si nécessaire.

## Technologies

- HTML5
- CSS3
